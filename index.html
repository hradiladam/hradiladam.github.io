Welcome.

Adam Hradil — a junior QA automation tester and developer with hands-on experience in modern testing tools like Postman, Playwright, Jest, Vitest, and pytest.


I build and test real applications using JavaScript/TypeScript, React, HTML, CSS, Node.js/Express, and Python, with a focus on writing clear, practical code and realistic automated tests.


I've created:

  • A simple Spotify Playlist Creator made with React, tested end-to-end using Playwright, and also unit, component, and integration tests with Vitest for the frontend and serverless backend functions hosted on Netlify.
  • A precision full-stack calculator app with both frontend UI and backend logic, tested end-to-end using Playwright, and also unit and integration tests with Jest for the frontend and backend.
  • A MySQL-based task manager CLI app, tested with Pytest using a separate test database, with full coverage of all CRUD operations.
  • A number of stand-alone Playwright tests simulating real user interaction with third-party websites.

I'm comfortable using VS Code, Git, and AI tools like ChatGPT to work efficiently and continue learning. I'm excited to join a team where I can continue growing and contribute meaningfully to test automation and product quality.
